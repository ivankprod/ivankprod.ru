# ivankprod.ru
Сайт IVANKPROD.RU
(go, webpack)

# Setup
1. install
```shell
./install.sh
```
2. build to /build_(dev|prod) dir
```shell
# development
./build.sh dev

# production
./build.sh prod
```

3. generate sitemap.xml for (dev|prod) build
```shell
# development
./sitemap.sh dev

# production
./sitemap.sh prod
```

4. run
```shell
# development build
./run.sh dev

# production build
./run.sh prod
```
